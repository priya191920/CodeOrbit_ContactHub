# 🚀 CodeOrbit Contact Hub

A full-stack contact hub application developed as part of my **CodeOrbit Technologies Full Stack Development Internship**.

The application allows users to submit their contact details and messages through a contact form. The submitted data is sent to a Node.js and Express.js backend and stored in MongoDB.

## ✨ Features

- 🏠 Home page
- 📞 Contact page
- 📝 Contact form
- 👤 Name input
- 📧 Email input
- 💬 Message input
- 🔗 Frontend and backend integration
- 🗄️ MongoDB database storage
- 🌐 REST API for contact submissions
- ⚡ React + Vite frontend
- 🎨 Tailwind CSS styling
- 🔒 Environment variables protected using `.gitignore`
